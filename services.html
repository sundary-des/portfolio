Product Design Services

How I can help — from discovery and UX flows to build-ready UI and prototypes.

Contact me
UX Audit for Complex Products
Find what blocks users today and get a prioritised fix plan with clear, actionable recommendations.
Service description
A focused review of your existing product to identify what's slowing users down and where the experience breaks. I'll map the key journeys, evaluate information hierarchy, states, and usability friction, and turn findings into clear, actionable improvements.

This is ideal when the product "works" but feels heavy, confusing, or inconsistent — and you need a realistic plan to fix it.
What you'll get
- A prioritised list of issues (quick wins + high-impact fixes)
- Clear recommendations with examples (what to change and why)
- A practical next-step plan your team can execute
MVP Product Design
From idea to a clickable MVP: core flows, key screens, and a prototype ready for testing and build.
Service description
End-to-end product design for a new MVP: from defining the core user journeys to creating key screens and a clickable prototype. The focus is on shipping the smallest version that proves value without cutting essential UX.

Great for early-stage products or new initiatives that need clarity, speed, and a solid foundation for development.
What you'll get
- MVP scope and core flows (user journeys + key states)
- Wireframes → high-fidelity screens for the MVP
- A clickable prototype ready for testing and build
Information Architecture & UX Flows
Turn complex workflows into clear, step-by-step journeys with edge cases covered.
Service description
I structure complex products so they feel simple: navigation, content hierarchy, and step-by-step flows that users can follow confidently. We'll define roles, states, edge cases, and error/empty scenarios — so the experience holds up in real use.

Best when the product has many features, screens, or user types and needs a clear logic.
What you'll get
- A clear IA/navigation proposal (how the product is organized)
- User flows for key scenarios (including edge cases)
- UX rules for states: empty, error, loading, permissions
New Feature Design
Design new features that fit the product logic and improve the experience without adding clutter.
Service description
Design of new features that fit your current product logic and improve the experience without adding clutter. I'll align the feature with user goals, integrate it into existing flows, and define the necessary states for implementation.

Ideal for product teams scaling functionality while keeping UX consistent.
What you'll get
- Feature flow + integration into existing journeys
- Key screens and interaction details (states included)
- A build-ready design handoff in Figma
Usability Testing & Iteration
Run quick, practical user testing and translate insights into focused design improvements.
Service description
Lightweight, practical user testing for your product or prototype — followed by focused improvements. The goal is to validate assumptions, spot friction early, and iterate quickly without heavy process.

Works well before development, during MVP, or when a feature isn't performing.
What you'll get
- A simple test plan + moderated sessions (remote)
- Findings summary (patterns, pain points, quotes)
- Prioritized design improvements (what to fix first)

Polished, build-ready UI for key screens — modern, consistent, and aligned with your product's tone. I focus on clarity, hierarchy, and components that engineers can implement without guesswork.

AI Feature UX
Design AI-assisted flows that feel safe, understandable, and user-controlled.
Service description
Design of AI-assisted workflows that feel trustworthy and user-controlled. I focus on clear inputs/outputs, review/edit loops, regeneration, transparency, and strong error recovery — so AI helps users ship real work, not just "generate content."

Best for products adding AI into creation, admin, or decision workflows.
What you'll get
