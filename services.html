Services

UX Audit for Complex Products
Find what blocks users today and get a prioritised fix plan with clear, actionable recommendations.
+
A structured review of your existing product — I map friction points, identify where users drop off or get confused, and deliver a prioritised list of UX fixes with clear rationale. Ideal for products that have grown fast and need a reset before the next build cycle.
MVP Product Design
From idea to a clickable MVP: core flows, key screens, and a prototype ready for testing and build.
+
I work with you to define what the MVP needs to do, then design the core flows and key screens — enough to validate the idea, get stakeholder buy-in, or hand off to engineering. Fast, focused, and grounded in what real users actually need.
Product Discovery Sprint
Clarify the problem, audience, and scope — so you build the right MVP without guesswork.
+
Before any design or development, I help teams get aligned on what problem they're solving, who for, and why now. The output is a clear problem statement, user profile, and scoped feature set — so the team builds with confidence, not assumptions.
Information Architecture & UX Flows
Turn complex workflows into clear, step-by-step journeys with edge cases covered.
+
I map the full product structure — navigation, content hierarchy, key user journeys — and make sure every path makes sense before a single screen is designed. Especially valuable for enterprise tools, dashboards, and multi-role systems where complexity is the default.
New Feature Design
Design new features that fit the product logic and improve the experience without adding clutter.
+
I design new features end-to-end — from concept to final screens — making sure they integrate with the existing product in a way that feels natural, not bolted on. Edge cases, empty states, and error flows are included, not afterthoughts.
Usability Testing & Iteration
Run quick, practical user testing and translate insights into focused design improvements.
+
I plan and run lightweight usability tests — moderated or unmoderated — then synthesise findings into clear design actions. No lengthy reports, just the insights that matter and what to do with them.
High-Fidelity UI for Web Products
Polished, build-ready UI for key screens — consistent, modern, and easy to implement.
+
When the UX is solid and the product needs to look the part, I design high-fidelity screens that are ready to hand off. Component-based, responsive-ready, and built around your design system — or I'll help you create one.
AI Feature UX
Design AI-assisted flows that feel safe, understandable, and user-controlled.
+
AI features fail when users don't understand what the system is doing or why. I design AI-assisted interactions that are transparent, controllable, and actually useful — so users trust the feature instead of working around it.
Handoff & Build Support
Detailed handoff, team support, and design QA to ensure what ships matches the intent.
+
I prepare design files for development handoff — annotated specs, component documentation, and responsive notes — and stay available during implementation to answer questions and review builds before they ship.
